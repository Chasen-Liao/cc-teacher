# 随堂测验 - 变量与数据类型

## Q1: 变量命名
以下哪个是合法的变量名？
- A) `2name`
- B) `my-name`
- C) `my_name`
- D) `my name`

**答案**: C
**解释**: 变量名只能包含字母、数字和下划线，不能以数字开头，不能包含空格或连字符。

---

## Q2: 数据类型
`"123"` 和 `123` 有什么区别？

**答案**: 
- `"123"` 是字符串类型（string），可以包含任意字符
- `123` 是整数类型（int），可以进行数学运算

---

## Q3: 代码理解
```python
x = 5
y = "5"
print(x + x)
print(y + y)
```
以上代码输出什么？

**答案**:
```
10
55
```

**解释**: `x + x` 是整数加法 (5 + 5 = 10)，而 `y + y` 是字符串拼接 ("5" + "5" = "55")
