# Programming Basics

## Overview

This course covers the fundamental concepts of programming that apply across all languages.

## Curriculum

### Module 1: Variables and Data Types
- What are variables?
- Basic data types (int, string, boolean, etc.)
- Variable naming conventions

### Module 2: Control Flow
- If/else statements
- Loops (for, while)
- Switch/match statements

### Module 3: Functions
- Function definition
- Parameters and arguments
- Return values
- Scope

### Module 4: Data Structures
- Arrays/Lists
- Objects/Dictionaries
- Sets

### Module 5: Object-Oriented Basics
- Classes and objects
- Methods
- Inheritance

## Learning Goals

By the end of this course, students will:
- Understand how programs store and manipulate data
- Write conditional logic and loops
- Create and use functions
- Work with common data structures
